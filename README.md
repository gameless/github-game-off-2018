GitHub Game Off 2018
====================

Setup
-----

1. [Install Git.][git]
2. [Install NPM.][npm]
3. Install this repository:
   ```sh
   git clone https://github.com/gameless/github-game-off-2018.git
   cd github-game-off-2018
   npm install
   ```

Run
---

```sh
npm start
```

[git]: https://git-scm.com/downloads
[npm]: https://www.npmjs.com/get-npm
